这是基于风控数据分析的sql报表练习

一、风险监控模块
1、实时风险大盘：核心指标（如异常率、拦截率）的实时监控。
2、风险趋势日报/周报：展示风险指标（如欺诈率、坏账率）的波动趋势。
3、风险事件明细表：记录所有触发风控规则的具体事件详情

二、策略分析模块
1、策略效果评估表：分析各条风控规则的命中率、准确率、召回率。
2、规则命中明细表：展示每条规则命中的具体案例。
3、策略迭代对比表：对比新旧策略上线后的数据差异

三、资产质量模块
1、资产质量分布表：按风险等级（如A/B/C级）统计资产分布。
2、逾期率报表：展示不同账龄（Vintage）的逾期表现。
3、迁徙率报表：分析逾期状态（M1->M2）的转化情况。

四、反欺诈模块
1、欺诈案件统计表：按欺诈类型（如身份冒用、虚假申请）统计。
2、黑名单命中报表：展示黑名单库的拦截效果。
3、设备/行为异常报表：分析设备指纹、IP地址等维度的异常聚集。

五、运营管理模块
1、人工审核工作量表：统计审核人员的处理量、通过率、误拒率。
2、催收绩效报表：展示催收团队的回收效果。
3、数据质量监控表：监控数据源（如征信、三方数据）的覆盖率和稳定性。
